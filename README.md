# dot-cosai
> Enables [`archinstall`](https://github.com/archlinux/archinstall) to work with repos of [`CachyOS`](https://github.com/CachyOS/linux-cachyos)

```shell
pacman-key --init
pacman -Sy git
git clone -b dot-cosai https://github.com/h8d13/archinstall-patch.git

./cosai
```
Or one liner: `curl -L https://github.com/h8d13/archinstall-patch/blob/dot-cosai/cosai | bash`
